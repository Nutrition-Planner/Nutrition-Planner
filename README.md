🥗 NutriPlanner – AI-Powered Nutrition & Meal Planning Web App

📌 Project Overview

NutriPlanner is an AI-powered web application designed to predict a user’s daily calorie requirements and generate personalized meal plans based on individual health and lifestyle inputs. By leveraging two trained machine learning models, the system delivers intelligent nutrition recommendations through a clean and responsive web interface. The project demonstrates a complete end-to-end machine learning pipeline — from data preprocessing and model training to backend integration and real-time prediction display — showcasing the practical application of AI in health-tech solutions.

⸻

🚀 Key Features
	•	🔥 Accurate daily calorie requirement prediction
	•	🥗 Personalized meal plan generation based on user profile
	•	📊 Real-time prediction results displayed dynamically
	•	🌐 Responsive and user-friendly web interface
	•	🧠 Integration of multiple trained ML models into a live application

⸻

🧠 How It Works

NutriPlanner collects user inputs such as age, gender, height, weight, and activity level through a web form. These inputs are processed by a calorie prediction model to estimate daily energy requirements. The predicted calorie value is then used by a second machine learning model to generate a suitable meal plan aligned with the user’s nutritional needs. The final output is presented instantly on the webpage in an easy-to-understand format.
